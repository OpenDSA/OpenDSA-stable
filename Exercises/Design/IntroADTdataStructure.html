A data structure is:

"the implementation of an abstract data type."
  • "The definition of a data type."
  • "A function that defines how an implementation should be written."
  • "A subset of a type."
  • "Typically implementated as an array or a list."

The definition of a data type is its logical form.

An array could refer to a data type or an implementation. A list describes an implementation of a data type. In either case, it is not a data structure.

An ADT is a model of a type. It does not define how the data type is implemented.

A type is a collection of values and therefore, a subset of an abstract data type, which defines both the type and a set of operations on the type.
