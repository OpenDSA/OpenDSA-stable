What is the running time of Heapsort when the input is an array where all record values are equal?

\Theta(n \log n)
  • \Theta(n^2)
  • \Theta(\log n)
  • \Theta(n ^ n)

Heapsort has the same asymptotic cost in the best, average, and worst cases.
