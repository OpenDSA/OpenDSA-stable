Stack A has entries a, b, c (with a on top), Stack B is initially empty. When an entry is popped out of stack A, it can be printed immediately or pushed to stack B. When an entry is popped out of stack B, it can only be printed. Which of the following permutations of a, b, c is not possible to print?

"c a b"
  • "b a c"
  • "b c a"
  • "a b c"
